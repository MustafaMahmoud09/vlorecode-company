### Supported Elements
All elements with `data-i18n` attributes are automatically translated:
```html
<!-- Example 1: Simple text -->
<h2 data-i18n="services.title">Our Services</h2>

<!-- Example 2: Nested content -->
<p data-i18n="portfolio.project1.description">Full-stack e-commerce...</p>

<!-- Example 3: Form labels -->
<label data-i18n="contact.name">Your Name</label>
```

## 📝 Adding New Translations

To add new translatable content:

1. **Add to HTML** with `data-i18n` attribute:
```html
<h3 data-i18n="newSection.title">New Title</h3>
```

2. **Add to `translations/en.json`**:
```json
{
  "newSection": {
    "title": "New Title in English"
  }
}
```

3. **Add to `translations/ar.json`**:
```json
{
  "newSection": {
    "title": "عنوان جديد بالعربية"
  }
}
```

4. **Refresh** - It works automatically!
